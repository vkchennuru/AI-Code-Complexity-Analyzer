Metric methodology (as described in paper Section 3)

Halstead Difficulty (D)

Measures how difficult code is to write or understand, based on the ratio of unique operators to operand usage. Rises sharply with type hints, lambda expressions, and advanced operators.

D = (η₁/2) × (N₂/η₂)

Halstead Effort (E)

The primary metric in this paper. Represents the total mental work (in elementary discriminations) required to process and evaluate a code fragment. Directly models extraneous cognitive load.

E = D × V = D × N × log₂(n)

Cognitive Complexity

Counts structural complexity: nesting depth, boolean chains, comprehensions, lambdas, and control flow. Each nesting level adds a penalty, reflecting how deeply a reader must track program state.

CC = Σ (1 + nesting_depth)
